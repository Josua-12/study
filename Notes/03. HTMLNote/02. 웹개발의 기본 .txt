1. 웹 개발의 기본
    1) HTML : 웹 문서의 뼈대를 만듦
    2) CSS : 웹 문서를 꾸밈
    3) 자바스크립트 : 사용자 동작에 반응하는 부분

2. 자바스크립트의 라이브러리(기능들을 호출해서 쓰는것)와 프레임워크(기능 추가)
    1) 제이쿼리(jQuery)
    2) 부트스트랩 (Bootstrap)
    3) 테일윈드(tailwindcss)
    4) 리액트 (React)
    5) 뷰(Vue)

3. 웹 개발환경
    1) 웹 브라우저 
        - 크롬, 엣지, 파이어폭스
    2) 웹 편집기
        - VS code
    3) 마크업하기 (MarkUp)
        - 웹 문서의 각 부분이 어떻게 표시되어야 하는지 내용마다 코드를 붙이는 것
        - 내용마다 붙이는 코드를 태그(tag)라고 한다.

4. HTML 기본 문서 만들기
    1) HTML
        - HyperText Markup Language
        - 웹 문서를 만드는 언어
        - HyperText: 문서를 서로 연결해주는 링크
    2) HTML 문서의 기본 구조
        - <!DOCTYPE html> 
            - 현재 문서가 HTML 언어로 작성한 웹 문서라는 뜻. 문서타입을 선언하는 것
            - 현재 표준은 HTML5
        - <html> ~ </html> 
            - 웹 문서의 시작과 끝을 나타내는 태그
        - <head> ~ </head>
            - 웹 브라우저가 웹 문서를 해석하는 데 필요한 정보. 데이터에 대한 정보
        - <meta>
            - HTML 문서(웹페이지)에 관한 정보(제작자, 내용, 키워드 등)
              검색엔진이나 브라우저에게 제공 
        - <body> ~ </body>
            - 웹 브라우저 화면에 나타나는 내용. 데이터에 대한 실제 내용
    3) 태그, 기억해 두면 좋은 4가지
        - 태그는 소문자로 작성을 권장
        - 여는 태그와 닫는 태그를 정확히 입력해야 함
        - 적당하게 들여쓰기를 해야 일목요연하고 사후관리가 쉬워짐
        - 일부 태그는 속성과 함께 사용함 
           ex: <html lang="ko"> <meta charset="UTF-8"> <div id="logo"> 
           id는 모든 태그에 속성으로 사용 가능
           인코딩은 문자로 바꾸는 것 디코딩은 코드로 바꾸는 것
    
5. 시맨틱(semantic) 태그
    1) 문서 내용에는 영향을 주지 않으면서 웹 브라우저가 문서 구조를 파악하는 데 
       중요한 역할을 하는 태그
    2) 주요 시멘틱 태그
        - <header> : 헤더 영역을 나타내는 태그
        - <nav> : 내비게이션 영역을 나타내는 태그
        - <main> : 핵심 콘텐츠를 담는 태그
                   웹 문서에서 한 번만 사용할 수 있다 
                   웹 문서마다 다르게 보여 주는 내용으로 구성함
        - <article> : 독립적인 콘텐츠를 담는 태그
                      웹 문서에서 여러 개 사용할 수 있다.
                      이 안에 <section> 태그를 넣을 수 있다.
        - <section> : 웹 문서에서 콘텐츠 영역을 나타내는 태그
                      몇 개의 콘텐츠를 묶을 때 사용
        - <asdie> : 사이트 바 영역을 나타내는 태그
        - <footer> : 푸터 영역을 나타내는 태그
                     ex) 사이트 제작 정보, 저작권 정보, 연락처 등

6. HTML 주석 달기
    1) comment, 설명글
    2) <!-- 주석 내용 -->

7. 정리
    1) (개발자 도구에서) Element(요소) ≒ Tag(태그)
        - 기본형태
            - <태그이름> 내용 </태그이름>
            - <태그이름>
                 내용 
              </태그이름>
    2) 속성(Attribute)과 값(Value)
        - <태그이름 속성="값"> 내용 </태그이름>
        - 태그(요소)의 기능을 확장하기 위해 '속성'을 사용함
    3) 부모 요소와 자식 요소
        - <부무요소>
            <자식요소></자식요소>
          </부모요소>
        - 요소의 내용으로 새로운 태그를 넣었을 때 '부모와 자식'의 관계가 생김
    4) 빈 태그(empty tag) : 종료 태그가 없는 태그
        - <태그이름> 또는 <태그이름/>  (html의 버전상 다를 수 있음)
        - 단, 속성은 있을 수 있음
          <태그이름 속성="속성값">
        - '빈태그'의 경우 종료태그가 없어서 내용을 넣을 수 없음. 
          대부분의 빈태그는 속성과 값이 필요함
    5) 웹 브라우저가 인식할 문서의 범위를 설정
        - <html></html> : 문서 전체 범위
        - <head></head> : 문서의 정보를 나타내는 범위
        - <body></body> : 문서의 구조를 나타내는 범위
