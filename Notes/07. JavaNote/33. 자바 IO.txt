1. 스트림 (stream)
    1) 자료가 오고갈 수 있는 가상의 데이터 통로
    2) 네트워크나 파일과 같은 데이터 흐름을 물의 흐름에 비유한 개념
    3) 자료를 읽어오는 입력 소스(source)와, 자료를 출력하는 대상(target)에 따라 각각 다른 스트림 클래스 제공됨

2. 스트림의 종류
    1) 입력 스트림과 출력 스트림
        - 스트림은 단방향으로 데이터가 흐름
        - 입력과 출력은 각각 별도의 스트림으로 처리되며, 동시에 하나의 스트림에서 입출력을 모두 수행할 수는 없음
        - 입력 스트림
            - InputStream (바이트 기반)
            - Reader (문자 기반)
        - 출력 스트림
            - OutputStream (바이트 기반)
            - Writer (문자 기반)
    2) 바이트 단위 스트림과 문자 단위 스트림 
        - 바이트 스트림
            - 자바의 기본 스트림은 원래 바이트(byte) 단위로 데이터 입출력을 처리함
            - 그림, 동영상, 음악 파일처럼 이진 데이터를 다룰 때 사용
            - 대표 클래스 : ~~InputStream, ~~OutputStream
        - 문자 스트림   
            - 문자 하나를 나타내는 char형은 2바이트임
            - 텍스트 파일처럼 문자 데이터를 다룰 때 사용
            - 대표 클래스 : ~~Reader, ~~Writer
    3) 기반 스트림과 보조 스트림
        - 기반 스트림
            - 데이터의 입출력 대상(파일, 키보드 등)에 직접 연결되어 있음
            - 읽고 쓰는 기능을 직접 제공함
        - 보조 스트림   
            - 직접 입출력을 처리하지는 않음
            - 다른 스트림(기반 스트림 등)을 감싸서 기능을 보조하거나 확장함
            
        - 예)
            기반 스트림 + 보조 스트림A + 보조 스트림B
            FileInputStream + BufferedInputStream + DataInputStream

3. 표준 입출력
    1) 표준 출력 스트림 
        - 변수 이름 : out 
        - 자료형 : static PrintStream 
    2) 표준 입력 스트림     
        - 변수 이름 : in 
        - 자료형 : static InputStream 
    3) 표준 오류 출력 스트림    
        - 변수 이름 : err 
        - 자료형 : static OutputStream

4. 바이트 단위 스트림
    1) InputStream  
        - 바이트 단위로 읽는 스트림의 최상위 클래스
        - 추상 메서드를 포함한 추상 클래스 
        - FileInputStream, ByteArrayInputStream, FilterInputStream
        - 메서드    
            - int read()
                - 입력 스트림으로부터 한 바이트의 자료를 읽음
                - 읽은 자료의 바이트 수를 반환함
            - int read(byte[] b)
                - 입력 스트림으로부터 b[] 크기의 자료를 b[]에 읽음
                - 읽은 자료의 바이트 수를 반환함
            - int read(byte[] b, int off, int len) 
                - 입력 스트림으로부터 b[] 크기의 자료를 b[]의 off 변수 위치부터 저장하며 len만큼 읽음
                - 읽은 자료의 바이트 수를 반환함
            - void close()
                - 입력
