1. 접근 제한자 (access modifier)
    1) 종류
        - private : 모든 외부 호출을 막음
                    클래스 안의 속성과 기능을 숨길 때 사용
                    외부 클래스에서 해당 기능 호출 불가
        - (default) : 같은 패키지 안에서 호출은 허용함
                      패키지 안으로 속성과 기능을 숨길 때 사용
                      외부 패키지에서 해당 기능 호출 불가
        - protected : 같은 패키지 안에서 호출은 허용함
                      패키지가 달라도 상속 관계의 호출은 허용함
                      상속 관계로 속성과 기능을 숨길 때 사용
                      상속 관계가 아닌 곳에서 해당 기능 호출 불가
        - public : 모든 외부 호출을 허용

    2) private > (default) > protected > public
        - 순서대로 private이 가장 많이 차단하고,
        - public이 가장 많이 허용함

    3) 사용 위치
        - 필드, 메서드, 생성자에 사용됨
        - 클래스
