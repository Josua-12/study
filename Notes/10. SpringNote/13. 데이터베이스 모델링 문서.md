## 3. 논리적 모델링 (Logical Modeling - Relation Model)

### 3.1 정규화 과정

#### 정규화 전 초기 상태 (비정규화)
** 초기 단일 테이블 예시: **
```
주문 장부 {
    고객ID, 고객명, 나이, 직업,
    도서ID, 도서명, 저자, 가격, 
    구매수량, 구매날짜,
    리뷰내용1, 리뷰날짜1,
    리뷰내용2, 리뷰날짜2,
    이미지경로1, 이미지경로2
}
```

#### 제 1정규형 (1NF) (비정규화) - 원자값 보장
** 정의 ** : 모든 속성이 원자값(더 이상 나눌 수 없는 값)을 가지며, 반복 그룹이 없어야 ㅎ마

** 1NF 적용 후 **
```
   주문 장부 {
    고객ID, 고객명, 나이, 직업,
    도서ID, 도서명, 저자, 가격, 
    구매수량, 구매날짜,
    리뷰내용, 리뷰날짜,
    이미지경로
}
```
