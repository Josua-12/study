꽃사슴(Sika deer, Cervus nippon)

우리나라에서 사육되는 사슴의 59% 정도를 차지하고 있는 소형 사슴으로 몸에 흰 반점이 있습니다.

수컷만 뿔을 가지며 가지 끝이 넷이고 엉덩이에는 흰 얼룩이 있고,

뒷다리에는 부전선이 선명하며 계절번식을 하는 반추동물로 우리나라의 기후풍토에 잘 적응돼 있습니다.
